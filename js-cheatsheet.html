JavaScript Cheat Sheet

Filter JavaScript concepts:

  • Variables:
    let x = 5; const y = 'hello';
  • Functions:
    function greet() { alert('Hi!'); }
  • Arrow Functions:
    const sum = (a, b) => a + b;
  • Objects:
    const user = { name: 'Alex', age: 20 };
  • Arrays:
    const list = [1, 2, 3];
  • Array Methods:
    list.map(n => n * 2);
  • Conditionals:
    if (x > 5) { ... }
  • Loops:
    for (let i = 0; i < 5; i++) { ... }
  • Events:
    button.addEventListener('click', fn);
  • DOM Manipulation:
    document.getElementById('id');
  • Template Literals:
    `Hello ${name}`
  • JSON:
    JSON.parse(jsonStr);
  • Local Storage:
    localStorage.setItem('key', 'value');
  • Try/Catch:
    try { ... } catch (e) { ... }
  • SetTimeout/SetInterval:
    setTimeout(fn, 1000);
