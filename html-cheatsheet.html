HTML Cheat Sheet

This page includes a quick reference to essential HTML elements and attributes. Use it to refresh your memory while coding.

Common HTML Tags

  • <!DOCTYPE> – Defines the HTML document type.
  • <html> – Root element of an HTML document.
  • <head> – Contains meta info, links, title, scripts, etc.
  • <title> – Sets the browser tab title.
  • <meta> – Provides metadata like charset or viewport settings.
  • <link> – Links external CSS files.
  • <script> – Embeds or links JavaScript.
  • <body> – Main content of the HTML page.
  • <h1> to <h6> – Heading tags for titles and subtitles.
  • <p> – Paragraph text.
  • <a href=""> – Anchor tag to link to other pages or websites.
  • <img src="" alt=""> – Displays an image with alternative text.
  • <ul>, <ol>, <li> – Unordered/ordered lists with list items.
  • <table>, <tr>, <td>, <th> – Table structure for displaying data.
  • <form>, <input>, <textarea>, <button> – User input elements.
  • <div> – Container for block-level content.
  • <span> – Inline container for text or inline elements.
  • <br> – Line break (used within paragraphs or text).
  • <hr> – Horizontal rule to separate content.

Example

<a href="https://example.com">Visit Example</a>

This creates a clickable link to example.com.
