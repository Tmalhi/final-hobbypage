CSS Cheat Sheet

Here are some essential CSS properties with examples to help you style your webpages with flair!

  • color: Sets the text color.
    color: red;
  • background-color: Sets the background color.
    background-color: #222;
  • font-family: Sets the font.
    font-family: 'Courier New', monospace;
  • font-size: Sets the text size.
    font-size: 16px;
  • text-align: Aligns text.
    text-align: center;
  • margin: Adds space outside an element.
    margin: 10px;
  • padding: Adds space inside an element.
    padding: 15px;
  • border: Adds a border.
    border: 2px solid yellow;
  • border-radius: Rounds borders.
    border-radius: 10px;
  • box-shadow: Adds shadow around elements.
    box-shadow: 0 4px 8px black;
  • text-shadow: Adds shadow to text.
    text-shadow: 2px 2px purple;
  • display: Sets display type.
    display: flex;
  • justify-content: Aligns items in flexbox.
    justify-content: center;
  • align-items: Aligns flexbox items.
    align-items: center;
  • position: Positions elements.
    position: relative;
  • z-index: Sets stacking order.
    z-index: 10;
